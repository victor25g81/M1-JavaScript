//Ejercicio 78. ¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear páginas web. 
Permite definir la estructura y el contenido de una página web mediante el uso de etiquetas y atributos, 
que se utilizan para describir diferentes elementos en la página, como encabezados, párrafos, imágenes, 
enlaces, tablas, formularios, entre otros.

HTML es un lenguaje de programación muy utilizado en el desarrollo web y es la base para la creación de sitios
 web. A menudo se utiliza en combinación con otros lenguajes de programación como CSS (Cascading Style Sheets)
  y JavaScript para crear sitios web interactivos y atractivos visualmente.
